# NHANES Ophthalmology-Metabolic ML Project

This repository contains a Python-based NHANES analysis pipeline for vision impairment modeling, feature exploration, and validation.

## Project Structure

- `nhanes_imputed.csv` - Processed NHANES dataset used for training and evaluation.
- `nhanes_merged.csv` - Intermediate merged NHANES data (source may vary).
- `programs/` - Python scripts for exploration, synthesis, validation, and modeling.
- `training_data/` - NHANES training source datasets in XPT format.
- `validation_data/` - NHANES validation source datasets in XPT format.
- `validation_set/` - Validation outputs or prepared validation data.
- Generated plots and diagnostics in the project root.

## Key Scripts

### `programs/data_learning_comprehensive.py`
- Trains three main model families: Decision Tree, Random Forest, and XGBoost.
- Handles class imbalance with SMOTE inside cross-validation.
- Performs:
  - Full feature modeling (Run A)
  - Metabolic-only modeling (Run B)
  - Age-stratified analysis (Run C)
- Performs hyperparameter tuning for XGBoost.
- Computes optimized threshold based on F1.
- Creates evaluation plots including ROC and Precision-Recall comparisons.
- Compares SMOTE-based training against class-weighted XGBoost.

### `programs/data_exploration.py`
- Performs exploratory data analysis on `nhanes_imputed.csv`.
- Computes correlations, covariance, high-correlation feature pairs, and data quality checks.
- Generates a correlation heatmap.
- Creates BPX vs `AVG_VISUAL_ACUITY` scatterplots to inspect blood pressure relationships.

## Requirements

Install the required Python packages before running the scripts.

```bash
python -m pip install pandas numpy scikit-learn xgboost imbalanced-learn shap matplotlib seaborn scipy
```

## Usage

Run the main training and evaluation script from the repository root:

```bash
python programs/data_learning_comprehensive.py
```

Run the data exploration script:

```bash
python programs/data_exploration.py
```

## Notes

- The project uses `VISION_IMPAIRED` as the main binary target.
- The exploration script uses `AVG_VISUAL_ACUITY` for continuous correlation analysis.
- The training pipeline produces both full-feature and metabolic-only model comparisons.
